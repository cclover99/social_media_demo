## Installation:  
Clone the repo:
``` 
git clone https://github.com/ccloverdev/social_media_demo.git
```
Setup the server:
```
mysql -S SERVER_NAME -d DATABASE_NAME -U USERNAME -P PASSWORD -i "./schema.sql"
```
Run Development:
```
npm build && npm run dev
```
